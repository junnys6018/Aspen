# Basics

## Comments

Aspen uses C style comments.

```
// This is a line comment

/*
    this block comment
    can span multiple lines
*/
```

## Printing

All output in Aspen is done through the `print` statement. There is no way to read user input in Aspen.

```
print "Hello, world!";
```
